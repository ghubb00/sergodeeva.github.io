How to create a blog using Hexo on GitHub Pages

Natalya Kosenko's Blog
2017-12-16
hexo

I was originally planning to create a blog using Create React App. It was extremely easy to set up create-react-app, but “the devil is in the details”. So I decided to use Hexo framework instead, as did not want to spend much time googling stuff and figiring out the details.

Before I started, I did not have any experience with hexo, and had no idea about node and nvm. So below an instruction for beginners :)
